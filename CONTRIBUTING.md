# Contributing to Strudel Jam

Thanks for wanting to contribute! Here's how to get started.

## Development setup

You need Node.js 18+, Rust 1.77+, and the Tauri v2 system dependencies for your platform (see the [Tauri prerequisites guide](https://v2.tauri.app/start/prerequisites/)).

You also need the Strudel repo cloned as a sibling directory:

```
parent/
  strudel/       # git clone https://github.com/tidalcycles/strudel
  strudel-jam/   # this repo
```

Then:

```bash
cd strudel && npm install && cd ..
cd strudel-jam && npm install
npm run build:strudel
npx tauri dev
```

## Project structure

- `src/` - React frontend (TypeScript, Tailwind CSS)
- `src-tauri/` - Rust backend (Tauri v2)
- `patterns/` - Bundled Strudel patterns
- `scripts/` - Build scripts
- `public/strudel/` - Built Strudel REPL (gitignored, generated by `build:strudel`)

## Making changes

1. Fork the repo and create a branch from `main`
2. Make your changes
3. Run `npx tsc --noEmit` to check for type errors
4. Test with `npx tauri dev`
5. Open a pull request

## Adding patterns

Drop a `.js` file in `patterns/` with valid Strudel code. The first line should be a comment describing the pattern. Keep patterns self-contained (no external sample loading that might fail).

## Code style

- TypeScript for the frontend, Rust for the backend
- Tailwind for styling, no separate CSS files
- Keep components in `App.tsx` for now (it's a single-file app)
- Meaningful commit messages

## Reporting issues

Open an issue with:
- What you expected to happen
- What actually happened
- Your OS and app version
- Steps to reproduce
